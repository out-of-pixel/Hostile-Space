# Refereeing the Game

This section provides guidance for Referees running Cepheus Engine games.

## The Referee's Role

The Referee is responsible for:

1. **Arbitrating the Rules** - Deciding how rules apply
2. **Creating the Universe** - Designing worlds and environments
3. **Controlling NPCs** - Playing non-player characters
4. **Adjudicating Actions** - Determining outcomes of character actions
5. **Maintaining Pacing** - Keeping the game moving
6. **Mediating Disputes** - Resolving player conflicts fairly
7. **Telling Stories** - Creating memorable narratives

## Before the Game

### Preparation

**Setting:**
- Establish the campaign universe
- Create worlds and locations
- Develop major factions and NPCs
- Define house rules

**Adventure:**
- Design encounters and challenges
- Create NPCs and their motivations
- Prepare descriptions
- Have backup encounters ready

**Materials:**
- Print character sheets
- Gather dice and miniatures
- Create reference sheets
- Make maps and handouts

### Session Zero

Hold a session before play starts to:
- Explain the setting
- Review key rules
- Discuss expectations
- Create characters together
- Establish tone and themes

## During Play

### Description and Immersion

Paint vivid pictures:
- Describe scenes in sensory detail
- Use voices and accents for NPCs
- Create atmosphere with descriptions
- Engage all senses

### Ruling on the Spot

When uncertain:
- Make a quick decision
- Keep the game moving
- Explain your reasoning
- Correct later if needed
- Stay consistent

### Difficulty Adjustments

Don't just use default difficulties:
- Easy tasks might not require rolls
- Routine tasks use +2 difficulty
- Consider circumstances
- Reward creative problem-solving
- Punish recklessness appropriately

### Failure is Interesting

When players fail:
- Describe consequences
- Don't simply say "no"
- Create new complications
- Allow alternative solutions
- Make failure advance the story

### Pacing

Control the game's speed:
- Don't rush important moments
- Skip boring downtime
- Use time jumps when appropriate
- Build tension with pacing
- Know when to move forward

## Non-Player Characters

### Creating NPCs

Quick NPC creation:
- Give them a distinctive trait
- Establish their motivation
- Determine their stats/skills (if needed)
- Decide how they react to PCs

### Playing NPCs

Make NPCs interesting:
- Give them distinct voices
- Show their personalities
- Have consistent motivations
- Make them react to player actions
- Remember what they promised

### Patron NPCs

Patrons are crucial:
- They give the party jobs
- They advance the plot
- Make them memorable
- Give them quirks and flaws
- Use them sparingly so they matter

## Encounters

### Combat Encounters

Structure them:
1. Set the scene
2. Roll initiative
3. Run combat round by round
4. Describe results
5. Award experience

Vary encounter difficulty:
- Mix easy and hard encounters
- Some fights should be escapable
- Not every fight is to the death
- Terrain and tactics matter

### Social Encounters

Include social challenges:
- NPCs with conflicting goals
- Negotiations and persuasion
- Deception and intrigue
- Allow creative problem-solving
- Use skill checks for uncertain outcomes

### Exploration Encounters

Encourage exploration:
- Describe interesting locations
- Hide secrets and treasure
- Create mysteries to solve
- Use maps and handouts
- Reward curiosity

## The Campaign

### Campaign Arc

Create a larger story:
- Introduce plot elements gradually
- Escalate tension over time
- Allow player agency
- Adapt to player decisions
- Build toward climactic moments

### Recurring Elements

Create continuity:
- Recurring NPCs and villains
- Factions with ongoing agendas
- World-changing consequences
- Long-term plot threads
- References to past events

### Campaign Pacing

Balance short-term and long-term:
- Each session should feel complete
- Longer arcs provide depth
- Give players meaningful choices
- Remember past decisions
- Let the world evolve

## Handling Problems

### Disruptive Players

Address behavior directly:
- Talk privately
- Explain the problem
- Listen to their perspective
- Find a solution
- If needed, end the relationship

### Rules Disputes

Resolve fairly:
- Listen to both sides
- Make a ruling
- Look up rule later
- If you were wrong, mention it
- Be flexible with house rules

### Spotlight Hogging

Balance participation:
- Give each player spotlight time
- Ask quiet players questions
- Limit dominant players gently
- Design some scenes for each character
- Encourage cooperation

### Boredom or Disengagement

Keep players invested:
- Ask what they're interested in
- Add complications to slow scenes
- Increase stakes and tension
- Involve their characters more
- Create side quests for their interests

## Difficulty Guidelines

### Task Difficulty by Category

| Type | Difficulty |
|------|-----------|
| Simple daily task | Simple (+6) or Easy (+4) |
| Routine specialized task | Routine (+2) |
| Standard trained task | Average (+0) |
| Difficult specialized task | Difficult (-2) |
| Expert-level task | Very Difficult (-4) |
| Impossible-seeming task | Formidable (-6) |

### Skill vs. Difficulty

- Untrained character needs +6 difficulty to match trained
- Expertise makes impossible tasks possible
- Practice and learning improve characters
- Some tasks truly require skills

## Narrative Techniques

### Building Tension

- Describe environmental hazards
- Introduce complications
- Time pressure (ticking clock)
- Moral dilemmas
- Unexpected twists

### Revealing Information

- Let players discover clues
- Use NPC dialogue for exposition
- Plant red herrings
- Reward investigation
- Give information through play

### Consequences

Make actions matter:
- Good deeds earn allies
- Bad deeds create enemies
- Failure has costs
- Success opens new paths
- Remember past decisions

## Treasure and Rewards

### Monetary Rewards

Reward player success:
- Combat: 100-1000 Credits
- Significant challenges: 1000-10000 Credits
- Major quests: 10000+ Credits
- Adjust for difficulty and party size

### Non-Monetary Rewards

Variety keeps it fresh:
- Equipment and weapons
- Shelter and bases
- Allies and contacts
- Reputation and titles
- Information and secrets

### Experience

Track character growth:
- Award after significant events
- Major encounters: 1-3 points
- Completing goals: 2-5 points
- Creative problem-solving: Bonus points
- Training and learning: Automatic progression

## Refereeing Tools

### Random Tables

Create quickly:
- NPC generators
- Encounter tables
- Treasure tables
- Event tables
- Name tables

### Miniatures and Maps

Visualize action:
- Grid maps for combat
- Miniatures for positioning
- Tokens for NPCs
- Sketch maps for locations
- Help with spatial descriptions

### Notes and Records

Remember everything:
- Session notes
- NPC details
- World details
- Player characters' goals
- Unresolved plot threads

## Final Tips

- Trust your instincts
- Prioritize fun over rules
- Listen to your players
- Keep learning and improving
- Remember: it's a collaborative game
- Take breaks and rest
- Most importantly: have fun!
