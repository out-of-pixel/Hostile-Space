## Damage and Injury

| State | Effect |
|-------|--------|
| Unwounded | Normal |
| Wounded | 1+ characteristics damaged |
| Seriously Wounded | All 3 characteristics damaged |
| Unconscious | 2+ characteristics at 0 |
| Dead | All 3 characteristics at 0 |

### Healing

- **Natural Healing:** 1D6 + End DM per day (full rest)
- **Active Lifestyle:** 1 + End DM per day
- **First Aid:** 2x Effect characteristic points (within 5 min)
- **Surgery:** Restores points but requires serious wounds
- **Medical Care:** 2 + End DM + Med skill per day

## Initiative and Actions

**Initiative:** 2D6 + Dex DM

**Each Round (6 seconds):**
- One significant action
- OR one minor action + one minor action + one minor action
- Unlimited free actions
- Reactions as needed (costs Initiative)

## Weapons Quick Reference

### Damage Notation

- **3D6** means roll three six-sided dice
- Add Effect from attack roll
- Armor reduces total damage
- Result is applied to Endurance

## Common Skill Checks

### Combat Skills

- **Gun Combat:** 2D6 + Skill + Dex DM
- **Melee Combat:** 2D6 + Skill + Str or Dex DM
- **Gunnery:** 2D6 + Skill + relevant DM

### Physical Skills

- **Athletics:** 2D6 + Skill + Str or Dex DM
- **Survival:** 2D6 + Skill + Int or End DM
- **Vehicle:** 2D6 + Skill + Dex DM

### Mental Skills

- **Computer:** 2D6 + Skill + Int DM
- **Science:** 2D6 + Skill + Int DM
- **Navigation:** 2D6 + Skill + Int DM

### Social Skills

- **Liaison:** 2D6 + Skill + Cha or Soc DM
- **Carousing:** 2D6 + Skill + Soc DM
- **Bribery:** 2D6 + Skill + Soc DM
